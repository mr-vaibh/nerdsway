# NerdsWay


## Setup
### Installation
 - `pip install pipenv`
 - `pipenv shell`
 - `pipenv install`
 - Also don't forget to configure environment vars in production

### DB table creation
 - `python manage.py makemigrations`
 - `python manage.py migrate`

### Run
 - `python manage.py runserver`
